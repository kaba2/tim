Dependencies
============

[[Parent]]: tim.txt

This page lists the dependencies of TIM. The version number that 
follows the library name is the one that the prebuilt TIM Matlab was 
built against. This is useful to know in case a later version breaks 
backward compatibility.

External dependencies
---------------------

### [Pastel 1.4](http://kaba.hilvi.org/pastel-1.4.0)

TIM refers to Pastel's include files in the form
`pastel/sys/array.h`. Pastel is licensed under the
MIT license.

### [Boost 1.53](http://www.boost.org)

TIM refers to Boost's include files in the form
`boost/operators.hpp`.
TIM only uses the template code found in Boost and
thus requires no linking with object code.
Boost is licensed under the Boost license.

### [Remark 1.6](http://kaba.hilvi.org/remark-1.6.0)

The documentation for TIM is written in Remark. 
This library is not required to build TIM.
